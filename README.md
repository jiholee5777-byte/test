# Season Tetris 🌸❄️🍂☀️

**Season Tetris**는 계절별 배경과 블록 색상이 바뀌는 브라우저용 테트리스 게임입니다.  
한 줄을 제거할 때마다 점수가 올라가고, 화면이 흔들리며, 계절이 바뀔 때마다 블록 색상과 배경이 바뀌어 더욱 재미있게 플레이할 수 있습니다.

![Season Tetris Screenshot](screenshot.png) <!-- 스크린샷 추가 가능 -->

---

## 🎮 게임 특징

- 한 줄 제거 시 **10점** 추가
- 줄 제거 시 **화면 흔들림 효과**
- **20점 단위로 계절 변경**
  - 봄 → 여름 → 가을 → 겨울 → 반복
- 계절별 **배경색과 블록 색상 변경**
- 브라우저에서 바로 실행 가능 (HTML + JavaScript)
